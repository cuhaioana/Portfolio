# Portfolio - Ioana Cuha

## Projects

### Library Management
Desktop application for managing books, users and borrowing operations in a library system. Built using a layered architecture, applying OOP principles and Dependency Injection. Implements authentication and CRUD operations with SQL database integration.  
https://github.com/cuhaioana/ProiectLibrary



### Online Shop
Full-stack web application for managing products and users in an online shop. Built using a layered architecture with REST APIs, applying OOP principles and Dependency Injection. Developed backend services with Spring Boot and integrated a React frontend with a PostgreSQL database.  
https://github.com/cuhaioana/ShopBootRender



### PC Benchmark Program
C++ application for benchmarking system performance, focusing on CPU operations, memory transfer and parallel execution. Uses multithreading and Windows API to measure execution time, MOPS and memory bandwidth.  
https://github.com/cuhaioana/BenchmarkPCs
